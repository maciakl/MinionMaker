Minion Academy
===

The source code for [minion.academy](http://minion.academy) web app.

Deploying Updates
---

App is running on heroku so just do:

    git push heroku master

To get logs:

    heroku logs

Dependencies
---

- [Sinatra](http://sinatrarb.com)
- [HAML](http://haml.info)
- [Pickup](https://rubygems.org/gems/pickup)
